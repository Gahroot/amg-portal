Accessibility Statement

WCAG 2.1 AA Compliant
Last updated: January 2026

Our Commitment

Anchor Mill Group (AMG) is committed to ensuring digital accessibility for people with disabilities. We continually improve the user experience for everyone and apply the relevant accessibility standards.

Conformance Status

The AMG Portal aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1, Level AA. These guidelines explain how to make web content more accessible for people with disabilities.

Current Status: We have implemented comprehensive accessibility features including:
  • Full keyboard navigation support
  • Screen reader compatibility (VoiceOver, NVDA, JAWS, TalkBack)
  • Proper ARIA labeling and roles
  • Color contrast ratios meeting WCAG AA standards
  • Responsive design for all device sizes
  • Skip links for efficient navigation
  • Focus indicators for keyboard users

Keyboard Navigation

The AMG Portal is fully navigable using a keyboard. Users can access all functionality without requiring a mouse.

Global Keyboard Shortcuts

Shortcut Action
Tab Move focus to the next interactive element
Shift + Tab Move focus to the previous interactive element
Enter / Space Activate buttons, links, and controls
Escape Close dialogs, menus, and modals
Ctrl + K Open command palette / global search
Ctrl + B Toggle sidebar
Ctrl + / Show keyboard shortcuts help

Data Table Navigation

Shortcut Action
↑ / ↓ Navigate between rows
← / → Navigate between cells
Space Select/deselect row
Ctrl + A Select all rows
Home Go to first column
End Go to last column
Ctrl + Home Go to first row
Ctrl + End Go to last row

Form Navigation

Shortcut Action
Tab Move to next form field
Shift + Tab Move to previous form field
↑ / ↓ Navigate select dropdowns
Space Toggle checkboxes and radio buttons

Skip Links

Skip links are provided at the beginning of each page to allow keyboard users to bypass repetitive navigation and jump directly to:

  • Skip to main content - Jumps to the primary content area
  • Skip to navigation - Jumps to the sidebar navigation menu

To use skip links, press Tab immediately after the page loads. The skip links will become visible when focused.

Screen Reader Support

The AMG Portal is tested with and supports the following screen readers:

Platform Screen Reader

Android TalkBack

Visual Design

Color Contrast

All text and interactive elements maintain a minimum color contrast ratio of 4.5:1 for normal text and 3:1 for large text, meeting WCAG 2.1 Level AA requirements.

Focus Indicators

All interactive elements display a visible focus indicator when navigated via keyboard. The focus ring uses a high-contrast color that is clearly visible against all backgrounds.

Dark Mode

The portal supports both light and dark color schemes, automatically detecting your system preference. You can also manually toggle between modes in the user settings.

Technical Specifications

Accessibility features in the AMG Portal are built using:

  • Semantic HTML5 elements
  • ARIA (Accessible Rich Internet Applications) attributes
  • WAI-ARIA Authoring Practices
  • Radix UI primitives for accessible components
